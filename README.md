# ternary-spiral

`ternary-spiral` is a Rust library for simulating spiral wave dynamics arising from Rock-Paper-Scissors cyclic dominance on a 2D spatial grid, where each cell holds one of three ternary states (Rock = -1, Paper = 0, Scissors = 1), and the system evolves through local invasion rules that generate rotating spiral patterns, coexistence metrics, biodiversity indices, and invasion front detection — capturing the emergent self-organization characteristic of ecological and chemical oscillating-reaction systems.
